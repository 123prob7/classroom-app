
# Classroom App

Simple app built in reactjs



## Run Locally

Clone the project

```bash
  git clone https://github.com/123prob7/classroom-app
```

Go to the project directory

```bash
  cd classroom-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Features

- log in/sign up/log out
- Classes listed
- Add new class


## Tech Stack

**Client:** React, Redux, MUI, TailwindCSS, Toastify

**Server:** Node, Express
